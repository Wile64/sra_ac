## sra_osrw

Un petit script rapide fait pour Zivan TV afin qu'il lance rapidement via des bouttons les actions de course suivantes lors de championnat FSW :

- Formation lap
- Start 5 secondes
- Lancement Standing
- Sortie safety
- SC IN THIS LAP
- Fin Safety
- Fin les drapeaux
- Virtual sc on
- Virtual sc off
- START UNLAPPING PROCEDURE
- END UNLAPPING PROCEDURE
